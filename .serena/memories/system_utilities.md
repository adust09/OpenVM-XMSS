# System Utilities for Darwin (macOS)

## File System Commands
- `ls`: List directory contents
- `find`: Search for files/directories
- `grep`: Search text patterns in files
- `rg` (ripgrep): Fast grep alternative (if installed)
- `cat`: Display file contents
- `head`/`tail`: View beginning/end of files
- `mkdir -p`: Create directories recursively
- `rm -rf`: Remove files/directories (use with caution)

## Git Commands
- `git status`: Check repository status
- `git add`: Stage changes
- `git commit -m`: Commit with message
- `git push -u origin <branch>`: Push to remote
- `git checkout -b <branch>`: Create and switch branch
- `git pull`: Fetch and merge changes
- `git submodule update --init --recursive`: Initialize submodules

## Process Management
- `ps aux`: List running processes
- `kill`: Terminate process
- `top`/`htop`: Monitor system resources
- `lsof`: List open files

## Network
- `curl`: Transfer data from/to servers
- `netstat`: Network statistics
- `ping`: Test network connectivity

## macOS Specific
- `open`: Open file/URL with default application
- `pbcopy`/`pbpaste`: Copy/paste with clipboard
- `caffeinate`: Prevent system sleep
- `mdls`: Display metadata attributes

## Development Tools
- `cargo`: Rust package manager and build tool
- `rustc`: Rust compiler
- `rustup`: Rust toolchain manager
- `code`: VS Code (if installed)

## Performance Monitoring
- `time`: Measure command execution time
- `du -sh`: Check disk usage
- `df -h`: Check disk space
- `Activity Monitor`: GUI system monitor

## Environment
- `echo $PATH`: Display PATH variable
- `which`: Locate command binary
- `env`: Display environment variables
- `export`: Set environment variables

## File Permissions
- `chmod`: Change file permissions
- `chown`: Change file ownership
- `ls -la`: List with permissions

## Archive Tools
- `tar`: Archive files
- `zip`/`unzip`: Compress/decompress files
- `gzip`/`gunzip`: GNU compression
